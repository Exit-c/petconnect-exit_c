반려동물 커뮤니티 프로젝트

**#사용한 기술**

React, Redux, typescript, styledComponent, firebase

**#기능 설명**

Firebase의 Authentication, Firestore, Storage 를 활용하여 구현

- 로그인(email, 소셜로그인: 구글, 깃허브) 및 로그아웃
  
- 회원가입 및 회원탈퇴

- 마이페이지 닉네임 변경

- 커뮤니티 글 등록(카테고리, 글 작성, 이미지업로드)

- 커뮤니티 글 정렬(최신순, 내가 작성한 글)

- 내가 작성한 글 삭제기능

- 커뮤니티 게시글 검색 기능
